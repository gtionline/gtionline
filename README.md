## Deployment

### Install Deployment Server
On the deployment server, the deployed vue-cli based website can be runned locally. The installation needs **root** permissions.
```
npm install -g serve
```
### Deploy Website
```
npm run build
```
### Run deployed website
```
serve -s dist
```
